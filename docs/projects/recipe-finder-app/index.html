Recipe Finder App

Alex Johnson

Search any ingredient and get recipe ideas instantly.

Repo: https://github.com/alexjohnson/recipe-finder

Demo: https://recipe-finder.vercel.app

  • React
  • Node.js
  • Express
  • REST API

Recipe Finder lets you type in ingredients you already have at home and returns matching recipes from a public API. Built with React on the frontend and an Express proxy to keep API keys server-side.
